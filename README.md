This is a simple Rock Paper Scissors game built using HTML, CSS, and JavaScript. The player competes against the computer, and the first to reach 3 points wins the game.

Features

Random computer choice each round

Score tracking for player and computer

Displays round results and winner message

Reset button to restart the game after a win

How It Works

The player selects Rock, Paper, or Scissors.

The computer randomly selects an option.

The round winner is determined based on the game rules.

Scores are updated after each round.

When either the player or computer reaches 3 points, the game ends and a winner is shown.

The game can be restarted using the Reset Game button.
